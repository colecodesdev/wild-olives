# Wild Olives Website (React Rebuild)

## Overview

A static marketing site for Wild Olives, a 30A bistro in Santa Rosa Beach, FL, rebuilt from Wix to a maintainable React SPA hosted cheaply on AWS.

  <img src="docs/app-screenshot.png" width="800" alt="screenshot of wild olives website">
</p>

---

## Tech Stack

- React
- Vite
- TailwindCSS
- React Router
- AWS S3 + CloudFront

---

## Run Locally

```
git clone https://github.com/colecodesdev/wild-olives.git
cd wild-olives
npm install
npm run dev
```
